# Transformer

Implementation workspace for the Transformer paper.

## Contents

- `model/transformer.py`: core implementation work
- `requirements.txt`: project dependencies

## Setup

Create a local virtual environment and install dependencies:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Notes

- Keep the virtual environment local to your machine and out of git.
- Prefer storing project-specific setup instructions in this README as the implementation evolves.
